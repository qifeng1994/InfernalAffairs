# InfernalAffairs
A game demo inspried by the film called InfernalAffairs. 

Developed based on UE 4.27.

Used asset from marketplace:Soul City, Third Person Story

Playing video:https://youtu.be/w1Cyx0QENxU

Player operations:
1. move:W A S D
2. interaction:F
3. start dialog:Space
4. open novel:Tab
